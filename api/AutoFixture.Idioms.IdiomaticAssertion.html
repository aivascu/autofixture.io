Syntax
public abstract class IdiomaticAssertion : IIdiomaticAssertion
Remarks

IdiomaticAssertion provides default implementations of all methods of IIdiomaticAssertion, making sure that higher-order methods call into lower-order methods; e.g. that Verify(Assembly) calls Verify(Type[]) with all public types in the assembly.

Implementers can override the appropriate methods instead of creating an implementation of IIdiomaticAssertion completely from scratch.
