Interface IIdiomaticAssertion

Represents an encapsulation of an idiomatic unit test assertion based on Reflection types.

Namespace: AutoFixture.Idioms
Assembly: AutoFixture.Idioms.dll
Syntax
public interface IIdiomaticAssertion
Remarks

Some unit tests tend to be very repeatable, following common idioms. Some such idiomatic tests can be expressed as general methods based on their System.Type or other Reflection-based instances.
